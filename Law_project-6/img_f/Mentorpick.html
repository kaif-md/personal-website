Implicit Graph Matrix DFS (Difficulty - easymedium)

Description

Given a matrix dimensions (R x C) and a starting cell (si, sj), print visited positions using DFS. 8-Direction to follow must be in this order N, NE, E, SE, S, SW, W, NW from any given cell.

Prohibited Keywords

No prohibited keywords for this problem.

Input Format

Line 1: Test cases count T.

Each test case has four space separated integers

R C si sj

Output Format

Print the matrix based on filling order starting with 1 in first cell and then 2 and so on.

After every test case print blank line of output.

Constraints

1 <= T <= 10

1 <= R, C <= 100
